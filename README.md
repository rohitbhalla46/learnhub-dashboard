# LearnHub Dashboard

A modern learning dashboard built with Next.js, Tailwind CSS, Supabase, and Framer Motion.

## Features

- Add Courses
- Delete Courses
- Dynamic Progress Tracking
- Supabase Database Integration
- Responsive UI
- Modern Dashboard Design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Framer Motion

## Installation

```bash
npm install
npm run dev
